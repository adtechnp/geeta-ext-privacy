Privacy Policy for Bhagavad Gita Slok Fetcher

Effective Date: April 23, 2026

Bhagavad Gita Slok Fetcher is a Chrome extension that allows users to view verses (slokas) from the Bhagavad Gita by entering chapter and verse numbers.

Overview

This extension is designed with user privacy in mind. It operates entirely within the user's browser and does not collect or share personal data.

Data Collection and Usage

This extension does not collect, store, or share any user data.

  • No personally identifiable information is collected
  • No browsing history is accessed or tracked
  • No user input is stored, transmitted, or shared

Permissions Justification

This extension requests only the minimum permissions required for its core functionality. These permissions are used solely to enable the extension interface and do not grant access to sensitive user data.

Remote Code and External Requests

The extension does not load remote code and does not make any external API or network requests. All data is bundled locally within the extension package.

Third-Party Services

This extension does not use any third-party services, analytics tools, or tracking technologies.

Data Security

As no user data is collected or transmitted, there is no risk of user data exposure through this extension.

Changes to This Policy

This privacy policy may be updated if the extension's features or data practices change. Updates will be reflected on this page with a revised effective date.
